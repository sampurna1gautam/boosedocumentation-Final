Overview

The BooseApp_Level6Sem1 is a Windows Forms application that implements the BOOSE (Basic Object-Oriented Scripting Environment) programming language with graphical drawing capabilities. This documentation covers all classes, methods, properties, and exceptions in the application.

Methods

Method

Thrown when the end command does not match the expected opening block type (if, while, for, method).
Execute()
Method
Executes the end command, handling different behaviors for loops, conditionals, and methods.
For while loops, it jumps back to the loop condition. For for loops, it updates the loop control variable and determines if the loop should continue. For methods, it returns to the line after the method call.
